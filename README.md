## 📝 Assignment Overview

### Assignment 1 — Goodness of Fit & Multivariate Data Generation

**Topics Covered:**
- Generated multivariate data using R
- Tested Goodness of Fit using statistical tests (e.g., Mardia's test, Henze-Zirkler test)
- Visualized the generated data using suitable plots (scatter plot matrix, Q-Q plots, etc.)
- Selected a real-world dataset and tested its multivariate normality
- Interpreted results with supporting plots and observations
